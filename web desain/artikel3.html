Apa itu Photoshop

March 23, 2020Views : 3,487,497

Adobe Photoshop atau dikenal dengan Photoshop merupakan salah satu software editing foto atau bitmap yang dikembangkan oleh perusahaan Adobe Systems. Software ini pertama dirilis sekitar tahun 1990 tapatnya pada tanggal 19 Februari 1990. Software ini bersifat komersil jadi apabila sobat komputer ingin menggunakan software ini membutuhkan lisensi resmi dari pengembang aplikasi dalam bentuk serial key. Adobe Photoshop dirancang untuk dapat digunakan pada sistem operasi windows dan Mac OS. sementara apabila sobat komputer ingin menggunakan pada sistem operasi Linux membutuhkan sebuah perangkat lunak tambahan seperti CroseOver atau Wine. Fitur Photoshop juga sangat lengkap dan mampu untuk menerima beberapa model warna seperti RGB, CMYK, Lab Color, Grayscale, Bitmap dan Duotone. Sementara untuk format file, Photoshop mampu untuk membuka dan menyimpan gambar dengan format jpg, jpeg, PNG, gif, dan format rester lainnya. Akan tetapi Photoshop juga memiliki format file khusus yang menjadi ciri khas software ini, antara lain PSD, PSB dan PDD.
